# 🌴 Travel Sri Lanka Website

A simple and responsive travel website developed using **HTML**, **CSS**, and **JavaScript**.  
This project provides information about popular tourist destinations in Sri Lanka, helping users explore and plan their trips.

---

## 📌 About

This project is designed as a **travel guide website** that showcases:
- Tourist attractions in Sri Lanka
- Travel destinations and highlights
- Adventure locations and tours

The goal is to promote Sri Lanka tourism by providing useful information in a clean and user-friendly interface.

---

## ✨ Features

- 🌍 Home page with introduction to Sri Lanka
- 🏝️ Tourist destinations and highlights
- 🧭 Adventure and tour pages
- 🖼️ Image gallery
- 📱 Simple and responsive design
- 🎨 Clean UI using HTML & CSS
